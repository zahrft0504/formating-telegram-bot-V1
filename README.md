I recommend you also develop a PRD (Product Requirements Document) for your vibe codding project before jumping into coding with the assistance of AI. You can refer to this video for the general framework https://youtu.be/iLCDSY2XX7E?si=Aa9BTj8EF3jPeMp7

This code is completely reliant on the "prompt" you have to really refine the prompt and make sure it gives you your desired output constantly before embedding it into your formatting function.

feel free to adjust the prompt to achieve any form of text manipulation you want.

you can host this locally via the local.py file but i find that it's most useful when hosted (as a web service on render).
